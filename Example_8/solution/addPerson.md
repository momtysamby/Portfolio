# Метод PUT AddPerson

Добавление сотрудника в систему. Идемпотентный метод. Используйте этот метод, чтобы внести сотрудника в систему. Для сохранения его в БД, воспользуйтесь методом PUT SavePerson.

#### Адрес

*URL*/addPerson

#### Тело запроса

| Тип                                                 | Имя    | Описание            |
|-----------------------------------------------------|--------|---------------------|
| *Guid*                                              | user   | Guid пользователя   |
| *[PersonProps](/Example_8/solution/PersonProps.md)* | person | Данные пользователя |

#### Ответ

В ответе вернется значение Guid, которое можно использовать для сохранения сотрудника в системе.
